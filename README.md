# Data Scientist Portfolio -- Luis Gutierrez 

Analytics engineering and data science projects demonstrating measurement frameworks, statistical analysis 
and member behavior modeling applied to real-world business problems

## Projects 

1. A/B Test Analysis - Conversion rate experimentation
   Statistical experiment measuring  conversion rate lift between control  and treatment groups.
   - Chi-square significance testing
   - 10,000 simulated users
   - 18% conversion lift detected
   - Executive recommendation included
     
2. Netflix  content affinity 35-45 Male Segment modeling
   Genre affinity scoring model identifying top content for the 35-45 male demographic.
   - 6,131 movies analyzed
   - Weighted genre scoring model
   - Top 15 titles ranked and visualized
   - Strategic content acquisition recommendations
   
4. Netflix Kids Retention - 3D rewatch rate model
   3D visualization of rewatch rates across age groups and genre types to model family plan retention.
   - 1,301 kid titles analyzed
   - 3D scatter plot across 3 dimensions
   - Retention insight: toddler content anchors family plans
